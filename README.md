# NovatiDevWeb

Este proyecto es una landing page profesional desarrollada para NovatiDev.

## Características principales
- Animación de logo de entrada (splash)
- Sección hero con video destacado
- Sección "¿Por qué elegirnos?" con diseño atractivo
- Portafolio de trabajos realizados
- Formulario de contacto
- Diseño responsive y moderno

## Estructura del proyecto
- `index.html`: Página principal
- `estilo.css`: Estilos personalizados
- `Funcionalidad.js`: Lógica de animaciones y video
- Imágenes y video de muestra

## Instalación y uso
1. Clona el repositorio:
   ```bash
   git clone https://github.com/EyeFinish/NovatiDevWeb.git
   ```
2. Abre la carpeta en tu editor favorito.
3. Abre `index.html` en tu navegador.
